# OpenGrall

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

**Открытый фреймворк для воплощённого ИИ. Один мозг, много тел.**

OpenGrall — это модульная платформа для создания автономных роботов, управляемых большими языковыми моделями (LLM) и визуальными моделями (VLM). Фреймворк спроектирован для работы на слабом железе (смартфоны, Raspberry Pi), устойчив к деградации сенсоров и способен к эволюционному самообучению.

---

## 🧠 Философия

*   **Адаптация, а не точность.** Робот должен работать в реальном мире: грязный лидар, ослеплённая камера, динамическое окружение.
*   **Модульность.** Добавление нового сенсора или инструмента не требует переписывания ядра.
*   **Симбиоз LLM и TinyML.** LLM принимает стратегические решения, TinyML (на ESP32) обеспечивает безопасность и рефлексы с правом вето.
*   **Эволюция.** Робот учится на своих действиях, формирует стратегии и со временем перестаёт дёргать LLM для рутинных задач.

## 🏗️ Архитектура

```

open_grall/
├── core/                    # Ядро: SensorMemory, WeightCalculator, StrategyLearner
├── sensors/                 # Сенсоры: Lidar, VLM, Аудио, Ультразвук, Одометрия
├── memory/                  # Память: Диалоги, Маршруты, Эпизоды
├── vision/                  # Зрение: VLM-клиент, Визуальная память
├── orchestration/           # Инструменты: 16+ действий (движение, речь, поиск)
├── agents/                  # LLM-клиенты: Ollama (Vikhr), YandexGPT
├── config.py                # Единый конфигурационный файл
├── agent_v5.py              # Главный класс робота
├── server_v7.py             # WebSocket сервер (нервная система)
└── dashboard.html           # Дашборд отладки (FREEZE, трассировка LLM)

```

## 🚀 Быстрый старт

1.  **Склонируйте репозиторий:**
    ```bash
    git clone https://github.com/Ferum93/OpenGrall.git
    cd OpenGrall
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Настройте `config.py`:**
    *   Укажите модель LLM (рекомендуется Vikhr-2.0-1.5B).
    *   Настройте пути к Vosk, камере, серверу.

4.  **Запустите сервер (на смартфоне/ПК):**
    ```bash
    python server_v7.py
    ```

5.  **Запустите агента:**
    ```bash
    python agent_v5.py
    ```

6.  **Подключите ESP32** с прошивкой TinyML.

## 📦 Возможности «из коробки»

*   🗣️ **Голосовое управление** (Vosk, wake word «Гралл»).
*   👁️ **Визуальное восприятие** (VLM, детекция объектов, сцен).
*   🦾 **16+ инструментов** (движение, речь, поиск в интернете, запись маршрутов).
*   🧠 **Автономное целеполагание** (робот сам придумывает, чем заняться в простое).
*   📚 **Пять видов памяти** (сенсорная, эпизодическая, решений, маршрутов, визуальная).
*   🔒 **Безопасность** (TinyML с правом вето на ESP32).
*   🛠️ **Дашборд отладки** (FREEZE-режим, трассировка промптов LLM).
